# VS121-P HTTP/MQTT JSON メモ

VS121-P は HTTP URL または MQTT broker に人数計測系の JSON を送信する。SunDS IoT では `device_info.device_sn` を DeviceID として扱う。Device Name は `device_info.device` を参考名として保存する。

## 対応イベント

| event | report_type | 主な用途 |
| --- | --- | --- |
| Region People Counting | interval / trigger | リージョン別の人数、占有状態 |
| Dwell Time Detection | interval / trigger | リージョン別の滞在時間 |
| Line Crossing Counting | interval / trigger | 入退場カウント |
| People Flow Analysis | interval | A-D エリア間の人流 |

## DeviceID

| JSON path | 用途 |
| --- | --- |
| `device_info.device_sn` | DeviceID として使用 |
| `device_info.device` | Device Name の参考値 |
| `device_info.device_mac` | 管理用メタ情報 |
| `device_info.ip_address` | 管理用メタ情報 |

## 主要 key

| key | 日本語名 | 型 | 単位 |
| --- | --- | --- | --- |
| `current_total` | 現在人数 | number | 人 |
| `max_counted` / `Max_counted` | 最大人数 | number | 人 |
| `total_mapped_regions` | マップ済みリージョン数 | number |  |
| `numbering_regions[]` | リージョン番号 | number[] |  |
| `occupancy[]` | 占有状態 | boolean/number[] |  |
| `current_counted[]` | 現在人数 | number[] | 人 |
| `dwell_time_data[].region` | 滞在リージョン | number |  |
| `dwell_time_data[].max_dwell_time` | 最大滞在時間 | number | s |
| `dwell_time_data[].avg_dwell_time` | 平均滞在時間 | number | s |
| `dwell_time_data[].people_id` | 人物ID | number |  |
| `dwell_time_data[].duration` | 滞在時間 | number | ms |
| `in_counted` | 入場人数 | number | 人 |
| `out_counted` | 退場人数 | number | 人 |
| `capacity_counted` | 在室増減 | number | 人 |
| `total_data.in_cumulative_counted` | 累計入場人数 | number | 人 |
| `total_data.out_cumulative_counted` | 累計退場人数 | number | 人 |
| `total_data.capacity_cumulative_counted` | 累計在室増減 | number | 人 |
| `line_trigger_data.in` | 入場トリガー | number | 人 |
| `line_trigger_data.out` | 退場トリガー | number | 人 |
| `flow_data.A-A` - `flow_data.D-D` | A-D 人流 | number | 人 |

## 注意

`snapshot` は画像コードのため、現行の SunDS IoT では数値 metric として保存しない。Raw 原始データと payload codec は使用しない方針。
