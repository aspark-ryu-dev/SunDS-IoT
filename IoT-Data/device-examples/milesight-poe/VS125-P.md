# VS125-P HTTP/MQTT JSON メモ

VS125-P は HTTP URL または MQTT broker に人数計測、滞在時間、注視時間、デジタル入力の JSON を送信する。SunDS IoT では `device_info.device_sn` を DeviceID として扱う。Device Name は `device_info.device_name` を参考名として保存する。

## 対応イベント

| JSON block | 主な用途 |
| --- | --- |
| `line_trigger_data[]` | ライン通過人数。children / staff / group / total 別に in / out を持つ。 |
| `region_trigger_data.region_count_data[]` | エリア内の現在人数。total / children / staff 別に current 値を持つ。 |
| `region_trigger_data.dwell_time_data[]` | エリア滞在時間。人物 ID、性別、スタッフ判定、滞在開始/終了を持つ。 |
| `attention_region_trigger_data.region_attention_time_data[]` | 有効な閲覧トラック消失時の注視時間。 |
| `di_trigger_data` | デジタル入力のトリガー回数とイベント名。 |

## DeviceID

| JSON path | 用途 |
| --- | --- |
| `device_info.device_sn` | DeviceID として使用 |
| `device_info.device_name` | Device Name の参考値 |
| `device_info.device_mac` / `device_info.wlan_mac` | 管理用メタ情報 |
| `device_info.ip_address` | 管理用メタ情報 |

## 主要 key

| key | 日本語名 | 型 | 単位 |
| --- | --- | --- | --- |
| `line_trigger_data[].children.in` | 子供 入場人数 | number | 人 |
| `line_trigger_data[].children.out` | 子供 退場人数 | number | 人 |
| `line_trigger_data[].children.female_in` | 子供 女性入場人数 | number | 人 |
| `line_trigger_data[].children.female_out` | 子供 女性退場人数 | number | 人 |
| `line_trigger_data[].children.male_in` | 子供 男性入場人数 | number | 人 |
| `line_trigger_data[].children.male_out` | 子供 男性退場人数 | number | 人 |
| `line_trigger_data[].staff.in` | スタッフ 入場人数 | number | 人 |
| `line_trigger_data[].staff.out` | スタッフ 退場人数 | number | 人 |
| `line_trigger_data[].total.in` | 総入場人数 | number | 人 |
| `line_trigger_data[].total.out` | 総退場人数 | number | 人 |
| `region_trigger_data.region_count_data[].total.current_total` | 現在総人数 | number | 人 |
| `region_trigger_data.region_count_data[].total.current_female` | 現在女性人数 | number | 人 |
| `region_trigger_data.region_count_data[].total.current_male` | 現在男性人数 | number | 人 |
| `region_trigger_data.dwell_time_data[].duration` | 滞在時間 | number | ms |
| `region_trigger_data.dwell_time_data[].people_id` | 人物ID | number |  |
| `region_trigger_data.dwell_time_data[].children` | 子供判定 | boolean |  |
| `region_trigger_data.dwell_time_data[].staff` | スタッフ判定 | boolean |  |
| `attention_region_trigger_data.region_attention_time_data[].attention_time_ms` | 注視時間 | number | ms |
| `di_trigger_data.di_trigger_count` | DI トリガー回数 | number | 回 |
| `device_info.cpu.cpu_temperature` | CPU 温度 | number | °C |
| `device_info.cpu.cpu_usage` | CPU 使用率 | number | % |
| `device_info.ram.memory_usage` | メモリ使用率 | number | % |
| `device_info.storage.storage_usage` | ストレージ使用率 | number | % |
| `device_info.running_time` | 稼働時間 | number | s |

## 注意

`device_info`、`network_info`、`time_info`、`*_name`、`*_uuid` は管理用メタ情報として扱い、Dashboard の表示候補からは原則外す。Raw 原始データと payload codec は使わず、必要な数値 key だけを KeyCatalog / Definitions に追加する。
